<h1>Microsoft Intune Device Enrollment Lab</h1>

<h2>Description</h2>

This project showcases the enrollment and management of a Windows device using Microsoft Intune and Microsoft Entra ID (Azure AD), simulating a real-world enterprise device management environment.

In this lab, I connected a Windows device to an organizational account, joined it to Microsoft Entra ID, and configured security features including Multi-Factor Authentication (MFA) using Microsoft Authenticator and Windows Hello PIN.

I then verified successful device enrollment and compliance status within the Microsoft Intune Admin Center, demonstrating modern device management (MDM) and endpoint security practices.

This project highlights hands-on experience with cloud-based device management, identity integration, and security configuration—demonstrating practical skills applicable to Help Desk and IT Support roles.

<br />

<h2>Technologies Used</h2>

- <b>Microsoft Intune</b>
- <b>Microsoft Entra ID (Azure AD)</b>
- <b>Microsoft Authenticator (MFA)</b>
- <b>Windows Hello for Business</b>
- <b>Windows 10/11</b>
